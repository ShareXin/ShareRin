# ShareRin

#### Requirements
ruby  
gnome-screenshot  
[Twitter](https://github.com/sixohsix/twitter)  
PyGtk  


#### How to use
1. git clone
2. add keyboard shortcuts
3. Add in Twitter App details into `config.rb`
4. Replace locations in shell scripts with YOUR locations
